A generator which must yield a unique ID on each next()-call. Used for generating ID’s for request messages. Default: internal default generator yielding integers 1, 2, ...
parallel_notification_handling
Affects by which strategy each notification handler will be launched to handle each notification. One of 3 supported choices: None, bsonrpc.ThreadingModel.THREADS or bsonrpc.ThreadingModel.GEVENT. Default: None
parallel_request_handling
Affects by which strategy each request handler will be launched to handle each request. One of 3 supported choices: None, bsonrpc.ThreadingModel.THREADS or bsonrpc.ThreadingModel.GEVENT. Default: bsonrpc.ThreadingModel.THREADS
threading_model
Affects the concurrency implementation of the internal dispatcher and message stream decoder. One of 2 supported choices: bsonrpc.ThreadingModel.THREADS or bsonrpc.ThreadingModel.GEVENT. Default: bsonrpc.ThreadingModel.THREADS
